# CG4 — MY (고객)

## 개요
- 목적: 고객이 자신의 프로필, 사진, 동의 내역, 계정 설정을 관리한다
- 진입 경로: 하단 탭 'MY'
- 이탈 경로:
  - '프로필 편집' 탭 → CL5
  - '내 사진 관리' 탭 → CL6
  - '로그아웃' 탭 → A1 (세션 종료)
  - '회원 탈퇴' 탭 → 탈퇴 확인 Modal (인라인)

---

## 레이아웃 구조
- 네비게이션: 하단 탭 ('MY' 활성)
- 헤더: 있음 — "MY" 타이틀
- 주요 영역: 프로필 카드 + 메뉴 목록
- 하단 고정 영역: 없음

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 프로필 사진 | image | N | 아바타 (원형) | 없으면 기본 아바타; 탭으로 변경 불가 (CL5에서 편집) |
| 이름 | text | Y | 프로필 카드 제목 | — |
| 휴대폰 번호 | text | Y | 프로필 카드 서브텍스트 | 마스킹 표시 (010-****-1234) |
| 노쇼 횟수 | number | N | 프로필 카드 서브텍스트 | 1회 이상일 때만 표시 (P1) |
