# DM1 — 대안 일정 제안 (Modal)

## 개요
- 목적: 디자이너가 신청자의 희망 일정이 맞지 않을 때 다른 가능한 일정을 제안한다
- 진입 경로: DL3(신청자 상세) → '대안 일정 제안' 버튼 탭
- 이탈 경로:
  - '제안 발송' 버튼 탭 → Modal 닫힘 → DL3(대안 제안 중 상태) → DG2(신청 관리)
  - '닫기' 버튼 또는 배경 탭 → Modal 닫힘 → DL3(변경 없음)

---

## 레이아웃 구조
- 네비게이션: 없음 (Modal, 상단 우측 '×' 닫기 버튼)
- 헤더: 있음 — "대안 일정 제안" 타이틀 + 신청자 이름 서브텍스트
- 주요 영역: 일정 슬롯 선택 (DL2에서 등록한 잔여 슬롯 목록)
- 하단 고정 영역: 있음 — '제안 발송' CTA 버튼

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 신청자 이름 | text | — | 헤더 서브텍스트 | 읽기 전용 |
| 신청자 희망 일정 | date+time | — | 참고 텍스트 (상단 안내) | "고객이 요청한 일정: OO월 OO일 OO시" |
| 제안 일정 선택 | date+time | Y | 라디오 버튼 목록 (잔여 슬롯) | 모집글의 확정되지 않은 잔여 슬롯만 표시 |

---

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 제안 발송 | '제안 발송' 버튼 탭 | 고객에게 대안 일정 알림 발송 → Modal 닫힘 → DL3 상태 업데이트 → DG2 이동 | 슬롯 선택 완료 |
| 닫기 | '×' 버튼 또는 배경 탭 | Modal 닫힘 → DL3 유지 | — |

---

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 기본 | 잔여 슬롯 있음 | 슬롯 목록 라디오 버튼 |
| 슬롯 없음 | 모집글의 모든 슬롯이 마감됨 | "제안 가능한 일정이 없습니다" 안내 + 닫기 버튼만 표시 |
| 발송 중 | API 호출 중 | CTA 버튼 로딩 |
| 오류 | 발송 실패 | 토스트 오류 + Modal 유지 |
