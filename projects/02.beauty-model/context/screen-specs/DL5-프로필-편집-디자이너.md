# DL5 — 프로필 편집 (디자이너)

## 개요
- 목적: 디자이너가 자신의 소개, 살롱 주소, 포트폴리오 사진을 수정하여 고객에게 보여지는 프로필을 관리한다
- 진입 경로: DG4(MY) → '프로필 편집' 메뉴 탭
- 이탈 경로:
  - 저장 완료 → DG4(MY)
  - 뒤로가기 (변경사항 있을 경우 확인 Alert) → DG4

---

## 레이아웃 구조
- 네비게이션: 상단 바 (뒤로가기 + "프로필 편집" 타이틀)
- 헤더: 없음
- 주요 영역: 폼 (기본 정보 섹션 + 포트폴리오 이미지 섹션)
- 하단 고정 영역: 있음 — '저장' CTA 버튼

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 프로필 사진 | image | N | 아바타 + '변경' 버튼 | 탭 시 갤러리 열기 |
| 디자이너명 | text | Y | 텍스트 입력 필드 | 최대 20자 |
| 살롱/작업 공간 주소 | text | Y | 텍스트 입력 필드 | 최대 100자; 추후 지도 연동 예정 |
| 디자이너 소개 | text | N | 텍스트 입력 필드 (멀티라인) | 최대 300자 |
| 포트폴리오 이미지 | image[] | N | 이미지 업로드 그리드 (최대 10장) | CL1(상품 상세) 디자이너 섹션에 표시 안 됨 (MVP); 추후 확장 |
| 휴대폰 번호 변경 | text | N | 텍스트 + '변경' 버튼 | 변경 시 인증번호 재발송 플로우 |

---

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 저장 | '저장' 버튼 탭 | 프로필 업데이트 → DG4로 이동 + 성공 토스트 | 필수 필드 유효 |
| 저장 실패 (검증) | '저장' 버튼 탭 | 인라인 오류 표시 | 필수 필드 비어있음 |
| 프로필 사진 변경 | 아바타 탭 | 갤러리 열기 → 선택 → 업로드 | — |
| 번호 변경 | '변경' 버튼 탭 | 인증번호 발송 플로우 인라인 표시 | — |
| 이미지 추가 | '+' 셀 탭 | 갤러리 열기 | 현재 이미지 수 10장 미만 |
| 이미지 삭제 | 이미지 '×' 탭 | 삭제 확인 Alert → 확인 → 제거 | — |
| 뒤로가기 | 상단 뒤로가기 탭 | 변경사항 있으면 "저장하지 않고 나가시겠습니까?" Alert | — |

## 엣지케이스
- **이미 사용 중인 번호로 변경 시도**: 409 응답 → "이미 가입된 번호입니다"
- **프로필 사진 10MB 초과**: 선택 즉시 오류 안내, 업로드 차단

---

## 패턴 후보
- **이미지 업로드 그리드**: CL6(내 사진 관리), DL2(모집글 등록)과 동일 구조
- **인라인 번호 변경 폼**: CL5(프로필 편집, 고객)와 동일 구조
