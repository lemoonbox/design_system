# CL6 — 내 사진 관리

## 개요
- 목적: 고객이 신청 시 사용할 사진을 사전에 등록·삭제하여 관리한다
- 진입 경로: CG4(MY) → '내 사진 관리' 메뉴 탭 / CL2(신청하기)에서 '내 사진에서 선택' 탭
- 이탈 경로:
  - 뒤로가기 → CG4(MY)
  - CL2에서 진입한 경우: 사진 선택 완료 → CL2로 복귀 (선택한 사진 전달)

---

## 레이아웃 구조
- 네비게이션: 상단 바 (뒤로가기 버튼 + "내 사진 관리" 타이틀)
- 헤더: 없음
- 주요 영역: 사진 그리드 (3열) + 추가 버튼 (그리드 첫 셀 고정)
- 하단 고정 영역: CL2에서 진입한 경우 — '선택 완료' CTA 버튼 표시

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 등록된 사진 목록 | image[] | — | 3열 그리드 썸네일 | 최대 등록 수: 제한 없음 (MVP에서는 최대 10장 권장) |
| 사진 등록 일시 | datetime | — | 그리드 셀 하단 서브텍스트 (선택) | 최신 순 정렬 |

---

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 사진 추가 | '+' 셀 탭 | 기기 갤러리 열기 → 선택 → 업로드 | — |
| 사진 삭제 | 사진 롱프레스 또는 편집 모드 'x' 버튼 | 삭제 확인 Alert → 확인 → 사진 제거 | — |
| 사진 전체화면 보기 | 사진 탭 (CG4 진입 시) | 전체화면 뷰어 오버레이 | CG4에서 진입한 경우 |
| 사진 선택 | 사진 탭 (CL2 진입 시) | 선택 표시 (체크) 토글 | CL2에서 진입한 경우 |
| 선택 완료 | '선택 완료' CTA 탭 | CL2로 복귀 + 선택 사진 전달 | CL2 진입 + 1장 이상 선택 시 |
| 뒤로가기 | 상단 뒤로가기 탭 | 진입 경로에 따라 CG4 또는 CL2로 이동 | — |

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 기본 | 사진 1장 이상 | 그리드 + '+' 셀 |
| 빈 상태 | 등록된 사진 없음 | "등록된 사진이 없습니다" + 추가 유도 텍스트, '+' 버튼만 표시 |
| 업로드 중 | 사진 업로드 처리 중 | 해당 그리드 셀 로딩 표시 |
| 선택 모드 (CL2 진입 시) | CL2에서 진입 | 사진 탭 시 체크 표시, 하단 CTA 표시 |
| 오류 | 업로드 실패 | 토스트 오류 메시지 + 해당 셀 오류 표시 |

---

## 엣지케이스
- **파일 용량 초과 (10MB 이상)**: 선택 즉시 오류 안내, 업로드 차단
- **지원하지 않는 파일 형식**: jpg/png/heic 외 형식 선택 시 오류 안내
- **업로드 중 네트워크 단절**: 재시도 버튼 + 임시 저장 없음 안내
