# DG1 — 모집글 목록

## 개요
- 목적: 디자이너가 자신이 등록한 모집글의 현황을 관리하고 신규 모집글을 등록한다
- 진입 경로: 하단 탭 '모집글' / 로그인·가입 완료 후 자동 진입 (디자이너)
- 이탈 경로:
  - 모집글 카드 탭 → DL1(모집글 상세)
  - '+' 버튼 탭 → DL2(모집글 등록)

---

## 레이아웃 구조
- 네비게이션: 하단 탭 ('모집글' 활성)
- 헤더: 있음 — "모집글" 타이틀 + 우측 상단 '+' 신규 등록 버튼
- 주요 영역: 상태 필터 탭 + 모집글 카드 목록 (수직 스크롤)
- 하단 고정 영역: 없음 (하단 탭 고정)

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 기본 | 모집글 1건 이상 | 카드 목록 표시 |
| 로딩 | 조회 중 | 카드 스켈레톤 |
| 빈 상태 | 등록 모집글 없음 | "등록한 모집글이 없습니다" + '등록하기' 버튼 |
| 오류 | 네트워크 실패 | 오류 메시지 + 재시도 버튼 |

---

## 엣지케이스
- **신청 대기가 있는 모집글**: 카드에 뱃지 강조; DG2(신청 관리) 탭으로도 동일 내용 확인 가능
- **모집 인원 자동 마감 처리된 모집글 (P1)**: 상태가 자동으로 '마감'으로 전환, 알림 없이 카드 상태 업데이트

---

## 패턴 후보
- **빈 상태 컴포넌트**: CG1, CG2, DG2, DG3의 빈 상태와 동일 레이아웃 구조
- **상태 필터 탭**: CG2, DG2, DG3와 동일한 탭 필터 컴포넌트 재사용
