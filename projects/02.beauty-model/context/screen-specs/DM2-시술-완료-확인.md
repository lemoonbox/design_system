# DM2 — 시술 완료 확인 (Modal)

## 개요
- 목적: 시술 완료 처리 전 동의 저장 여부를 재확인하고 최종 완료 처리를 승인한다
- 진입 경로: DL4(예약 상세) → '시술 완료' 버튼 탭
- 이탈 경로:
  - '완료 확인' 버튼 탭 → Modal 닫힘 → DG3(완료 상태)
  - '닫기' 버튼 또는 배경 탭 → Modal 닫힘 → DL4(변경 없음)

---

## 레이아웃 구조
- 네비게이션: 없음 (Modal, 상단 우측 '×' 닫기 버튼)
- 헤더: 있음 — "시술 완료 처리" 타이틀
- 주요 영역: 완료 처리 안내 + 동의 내역 저장 확인
- 하단 고정 영역: 있음 — '완료 확인' 버튼 (Primary) + '닫기' 버튼 (Secondary)

## 엣지케이스
- **이미 완료 처리된 예약**: 서버 409 응답 → "이미 완료 처리된 예약입니다" 안내 후 Modal 닫힘 → DG3 새로고침
- **동의 내역 미저장 상태**: 시스템상 발생하지 않아야 하나, 저장 실패 이력 감지 시 "동의 내역을 확인할 수 없습니다. 고객센터에 문의하세요" 안내
