# DL2 — 모집글 등록/편집

## 개요
- 목적: 디자이너가 시술 상품 정보를 입력하여 새 모집글을 등록하거나 기존 모집글을 수정한다
- 진입 경로: DG1(모집글 목록) → '+' 버튼 (등록 모드) / DL1(모집글 상세) → '편집' 버튼 (편집 모드)
- 이탈 경로:
  - 게시 완료 → DL1(해당 모집글 상세)
  - 임시저장 → DG1(목록)
  - 모집글 삭제 → DG1(목록)
  - 뒤로가기 (변경사항 있을 경우 확인 Alert) → 진입 화면

---

## 레이아웃 구조
- 네비게이션: 상단 바 (뒤로가기 + 타이틀 "모집글 등록" 또는 "모집글 편집" + 우측 '임시저장' 텍스트 버튼)
- 헤더: 없음
- 주요 영역: 폼 (섹션별 입력)
- 하단 고정 영역: 있음 — '게시하기' CTA 버튼 (편집 모드: '저장하기') + 삭제 링크 (편집 모드만)

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 시술명 | text | Y | 텍스트 입력 필드 | 최대 30자 |
| 시술 가격 | number | Y | 숫자 입력 필드 (원 단위) | 최소 금액 제한 가능 (추후 결정) |
| 예약금 활성화 | boolean | Y | 토글 스위치 | ON: 예약금 30% 선결제 / OFF: 당일 전액 |
| 총 모집 인원 | number | Y | 숫자 입력 (스테퍼 또는 직접 입력) | 최소 1명 |
| 가능 일정 | date+time[] | Y | 날짜·시간 슬롯 추가 UI | 1개 이상 필수; 슬롯별 '추가' / '삭제' |
| 모집 조건 — 성별 | select | N | 세그먼트 선택 (제한 없음 / 여성 / 남성) | — |
| 모집 조건 — 기타 | text | N | 텍스트 입력 필드 | 최대 100자 |
| 시술 참고 이미지 | image[] | N | 이미지 업로드 (최대 3장) | — |

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 게시하기 | '게시하기' 버튼 탭 | 모집글 생성 + DL1으로 이동 | 필수 필드 완료 |
| 저장하기 | '저장하기' 버튼 탭 (편집 모드) | 모집글 수정 + DL1으로 이동 | 필수 필드 완료 |
| 게시/저장 실패 (검증) | 버튼 탭 | 미완료 필드 인라인 오류 표시 | 필수 필드 미완료 |
| 임시저장 | '임시저장' 버튼 탭 | 현재 입력값 저장 + DG1으로 이동 | 필수 필드 미완료도 가능 |
| 슬롯 추가 | '+ 일정 추가' 탭 | 날짜·시간 선택 피커 표시 → 슬롯 추가 | — |
| 슬롯 삭제 | 슬롯 행 '×' 탭 | 해당 슬롯 제거 | 이미 신청자가 있는 슬롯은 삭제 불가 (편집 모드) |
| 이미지 추가 | 이미지 업로드 영역 탭 | 갤러리 열기 | 현재 이미지 수 3장 미만 |
| 이미지 삭제 | 이미지 '×' 탭 | 해당 이미지 제거 | — |
| 모집글 삭제 | '삭제' 링크 탭 (편집 모드) | 삭제 확인 Alert → 확인 → DG1으로 이동 | 편집 모드 + 확정 예약 없는 경우 |
| 뒤로가기 | 상단 뒤로가기 탭 | 변경사항 있으면 Alert (임시저장 / 나가기 / 취소) | — |

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 등록 모드 | '+' 버튼에서 진입 | 빈 폼, CTA '게시하기' |
| 편집 모드 | DL1 '편집'에서 진입 | 기존 데이터 채워진 폼, CTA '저장하기' + 삭제 링크 |
| 제출 중 | API 호출 중 | CTA 버튼 로딩 |
| 오류 | 서버 오류 | 토스트 오류 메시지 |

## 엣지케이스
- **확정 예약이 있는 슬롯 삭제 시도**: "이미 예약이 확정된 일정은 삭제할 수 없습니다" 인라인 안내
- **확정 예약이 있는 모집글 삭제 시도**: "확정된 예약이 있어 삭제할 수 없습니다. 예약을 먼저 처리해 주세요" 안내
- **임시저장 후 재진입**: 기존 임시저장 데이터 복원 여부 확인 Alert ("이전에 저장된 내용을 불러오시겠습니까?")
