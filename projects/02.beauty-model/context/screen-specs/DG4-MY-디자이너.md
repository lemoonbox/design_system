# DG4 — MY (디자이너)

## 개요
- 목적: 디자이너가 자신의 프로필, 정산 내역, 계정 설정을 관리한다
- 진입 경로: 하단 탭 'MY'
- 이탈 경로:
  - '프로필 편집' 탭 → DL5
  - '로그아웃' 탭 → A1

---

## 레이아웃 구조
- 네비게이션: 하단 탭 ('MY' 활성)
- 헤더: 있음 — "MY" 타이틀
- 주요 영역: 프로필 카드 + 정산 요약 카드 + 메뉴 목록
- 하단 고정 영역: 없음

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 프로필 사진 | image | N | 아바타 (원형) | 없으면 기본 아바타 |
| 디자이너명 | text | Y | 프로필 카드 제목 | — |
| 살롱/작업 공간 주소 | text | Y | 프로필 카드 서브텍스트 | — |
| 정산 예정 금액 | number | N | 정산 요약 카드 | 정산 대기 중인 총액 |
| 최근 정산 완료 금액 | number | N | 정산 요약 카드 | 최근 T+3 정산 완료 금액 |
| 정산 내역 보기 링크 | — | — | 텍스트 링크 | 정산 요약 카드 하단 |

---

## 메뉴 목록

| 메뉴명 | 이동 대상 | 비고 |
|--------|----------|------|
| 프로필 편집 | DL5 | — |
| 정산 내역 | 정산 내역 화면 (MVP 미포함 또는 간이 표시) | 추후 확장 |
| 서비스 이용약관 | 외부 링크 또는 Modal | — |
| 개인정보 처리방침 | 외부 링크 또는 Modal | — |
| 로그아웃 | — | 확인 Alert 후 → A1 |
| 회원 탈퇴 | — | 확인 Modal 후 계정 비활성화 → A1 |

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
