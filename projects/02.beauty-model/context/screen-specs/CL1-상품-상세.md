# CL1 — 상품 상세

## 개요
- 목적: 시술 상품의 전체 조건과 디자이너 정보를 확인하고 신청 여부를 결정한다
- 진입 경로: CG1(상품 목록) → 상품 카드 탭
- 이탈 경로:
  - '신청하기' 버튼 탭 → CL2 (로그인 필수; 비로그인 시 A2 리다이렉트)
  - 뒤로가기 → CG1

---

## 레이아웃 구조
- 네비게이션: 상단 바 (뒤로가기 버튼, 제목 없음)
- 헤더: 없음 (참고 이미지가 상단을 차지)
- 주요 영역: 스크롤 (참고 이미지 → 시술 정보 → 디자이너 정보 → 시술 조건 → 가능 일정)
- 하단 고정 영역: 있음 — 가격 표시 + '신청하기' CTA 버튼

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 참고 이미지 | image[] | N | 상단 전체 너비 이미지 (최대 3장, 좌우 스와이프) | 없으면 기본 플레이스홀더 |
| 시술명 | text | Y | 본문 제목 (대형) | — |
| 모집 상태 배지 | select | Y | 배지 (모집 중 / 마감 임박 / 마감) | — |
| 시술 가격 | number | Y | 강조 텍스트 | — |
| 결제 방식 | text | Y | 서브텍스트 | "예약금 30% 선결제" 또는 "당일 전액 결제" |
| 취소 규정 요약 | text | — | 안내 텍스트 | "48시간 이내 취소 시 예약금 미환불" |
| 디자이너 프로필 사진 | image | N | 아바타 썸네일 | 없으면 기본 아바타 |
| 디자이너명 | text | Y | 디자이너 섹션 텍스트 | — |
| 살롱/작업 공간 주소 | text | Y | 디자이너 섹션 서브텍스트 | — |
| 디자이너 소개 | text | N | 디자이너 섹션 본문 | 없으면 섹션 미표시 |
| 모집 조건 — 성별 제한 | select | N | 텍스트 (없음 / 여성 / 남성) | 없으면 미표시 |
| 모집 조건 — 기타 | text | N | 텍스트 | 예: "펌 경험 없는 분 우대" |
| 모집 인원 | number | Y | 텍스트 | 예: "총 3명 · 2자리 남음" |
| 가능 일정 목록 | date+time[] | Y | 일정 칩 목록 (선택 불가, 열람용) | 신청 시 CL2에서 선택 |

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 신청하기 | '신청하기' 버튼 탭 | CL2로 이동 | 모집 중 상태 + 로그인 완료 |
| 신청 불가 | '신청하기' 버튼 탭 | 버튼 비활성 (탭 불가) | 마감 상태 |
| 비로그인 신청 시도 | '신청하기' 버튼 탭 | A2로 리다이렉트 (복귀 경로 저장) | 비로그인 |
| 이미지 전체화면 | 참고 이미지 탭 | 전체화면 뷰어 오버레이 | — |
| 뒤로가기 | 상단 바 뒤로가기 탭 | CG1으로 이동 | — |

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 기본 (모집 중) | 잔여 인원 있음 | CTA 버튼 활성 |
| 마감 임박 | 잔여 인원 1명 | "마감 임박" 배지 + CTA 활성 |
| 마감 | 인원 소진 또는 디자이너 수동 마감 | CTA 비활성 ("마감된 모집입니다"), 카드 dimmed |
| 로딩 | 데이터 조회 중 | 스켈레톤 |
| 오류 | 네트워크 실패 | 오류 메시지 + 재시도 버튼 |

## 엣지케이스
- **모집글 삭제된 상태로 진입**: 서버 404 응답 → "삭제된 모집글입니다" 안내 + CG1으로 이동 버튼
- **이미 같은 상품에 신청한 경우**: CTA를 "신청 완료" 비활성 버튼으로 변경, 예약 상태 링크 표시
- **가능 일정 전체 마감**: 인원은 남아있어도 슬롯 없으면 CTA 비활성

---

## 패턴 후보
- **시술 요약 카드**: CL2(신청하기), CL3(결제), CL4(예약 상세) 헤더와 동일 데이터 — 공통 패턴화 대상
- **전체화면 이미지 뷰어**: DL3(신청자 상세)과 동일 뷰어 컴포넌트 재사용
- **디자이너 정보 섹션**: DL1(모집글 상세)에서도 동일 구조 표시
