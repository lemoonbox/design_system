# CM2 — 동의 내역 확인 (Modal)

## 개요
- 목적: 신청 시 동의한 초상권·SNS·후기 활용 동의 내용 전문을 열람한다
- 진입 경로: CL2(신청하기) → 동의 항목 '내용 보기' 링크 탭 / CL4(예약 상세) → '동의 내역 보기' 링크 탭 / CG4(MY) → '동의 내역' 메뉴 탭
- 이탈 경로:
  - '닫기' 버튼 탭 또는 배경 탭 → Modal 닫힘 → 진입 화면으로 복귀

---

## 레이아웃 구조
- 네비게이션: 없음 (Modal, 상단 우측 '×' 닫기 버튼)
- 헤더: 있음 — 동의 항목명 (예: "초상권 및 SNS 활용 동의")
- 주요 영역: 동의 내용 전문 텍스트 (스크롤 가능)
- 하단 고정 영역: 있음 — '확인' 버튼 (CL2 진입 시 동의 체크 연결, 그 외는 단순 닫기)

---

## 데이터 필드

| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 동의 항목명 | text | Y | Modal 제목 | 예: "초상권·후기·SNS 활용 동의" |
| 동의 내용 전문 | text | Y | 스크롤 가능 본문 | DB에 저장된 동의 내용 텍스트 (법적 근거 포함) |
| 동의 일시 | datetime | N | 본문 하단 | CL4·CG4에서 진입한 경우 (기 동의 내역)만 표시 |

---

## 액션

| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 닫기 | '×' 버튼 또는 배경 탭 | Modal 닫힘 | — |
| 확인 (CL2 진입 시) | '확인' 버튼 탭 | Modal 닫힘 + CL2 동의 체크박스 자동 체크 | CL2에서 진입한 경우 |
| 확인 (그 외) | '확인' 버튼 탭 | Modal 닫힘 | CL4, CG4에서 진입한 경우 |

---

## 상태 정의

| 상태 | 발생 조건 | UI 처리 |
|------|----------|---------|
| 기본 | 동의 내용 로드 완료 | 전문 텍스트 스크롤 표시 |
| 로딩 | 내용 조회 중 | 텍스트 영역 스켈레톤 |

## 엣지케이스
- **동의 내용 버전 변경**: 신청 당시 버전과 현재 버전이 다를 경우 CL4·CG4 진입 시 "신청 당시 동의 내용을 표시합니다" 안내
