# M4 — 이송·케이스 완료 확인

## 개요
- 목적: 병원 도착 후 **이번 케이스를 완료할지** 확인하고, 완료 시 홈으로 돌아가게 한다.
- 진입 경로: (ia.md) L4에서 **병원 도착** 후 완료 확인 진입 — **트리거: L4(이송 중) 하단 “병원 도착”(또는 동등) CTA** 탭 시( L4 스펙과 정합).
- 이탈 경로: (ia.md) **완료** 확인 → 케이스 종료·**G1**; **취소·미완료** → **L4** 이송 중 유지.

## 레이아웃 구조
- 네비게이션: 모달.
- 헤더: 있음 — “이송을 완료할까요?” / “케이스를 종료할까요?” 등.
- 주요 영역: 선택 병원·도착 확인 문구; 미완료 시 **L4에서 계속** 가능함을 안내.
- 하단 고정 영역: 있음 — **아니오·계속** · **네·완료**(주요).
