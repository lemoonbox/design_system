# M3 — 출동 종료 경고

## 개요
- 목적: L1에서 상위(구급 홈 등)로 나가며 **지금 출동을 완료하는지** 확인해 기록 세션을 의도치 않게 끊지 않도록 한다.
- 진입 경로: (ia.md) **L1**에서 **G1 등 상위로 이탈 시도**(뒤로가기 등).
- 이탈 경로: (ia.md) **취소** → **L1** 유지; **확인(출동 완료)** → 출동 완료 처리 후 **G1**(또는 정책상 목적 화면).

## 레이아웃 구조
- 네비게이션: 모달(다이얼로그 권장 — 뒤로와 충돌 적음).
- 헤더: 있음 — “출동을 종료할까요?” 등.
- 주요 영역: **미반영·로컬만 저장 데이터**가 있을 수 있음을 경고(ia “미반영 데이터 경고”와 정합); 동기화 대기 안내.
- 하단 고정 영역: 있음 — **계속 기록(L1)** · **출동 종료하고 나가기**(파괴적 강조).

## 데이터 필드
| 필드명 | 타입 | 필수 여부 | 표시 방식 | 비고 |
|--------|------|----------|----------|------|
| 미전송 구간 수 | number | N | 텍스트 | 0이면 문구 완화 가능 |
| 마지막 저장 시각 | date | N | 텍스트 | |

## 액션
| 액션명 | 트리거 | 결과 | 조건 |
|--------|--------|------|------|
| 취소 / 계속 기록 | 보조·취소 | **L1** 유지 | 항상 |
| 출동 종료 확인 | 확인 버튼 | 출동 완료 처리 후 **G1** | 전송 실패 시 M2 가능 |
| 바깥 탭 | 시스템 | 취소와 동등 | 플랫폼 정책 |

## 상태 정의
